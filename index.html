How to push onto this website:

  • Make your changes
  • Go onto Git Bash
  • Go into BinBinBinBinBinBinBinBinBin.github.io using cd ./
  • Type git add .
  • Type git commit -m"[Your Message]"
  • Type git push
  • Check your website, it should be on there.
  • Make sure you do command shift r to hard reset the website

How to push locally

  • Make your changes
  • Go onto your local website/file
  • Press command shift r to hard reset the website
  • Your changes should now be there locally
  • Local means not public; nobody else can see your changes
